TYPES **************************************------------------------------------------

There are different types of joins available in SQL: INNER JOIN: returns rows when there is a match in both tables.- RETURN COMMON IN BOTH LEFT JOIN: returns all rows from the left table, even if there are no matches in the right table.- EVERYTHING FROM FIRST/LEFT, and COMMON FROM 2ND RIGHT JOIN: returns all rows from the right table, even if there are no matches in the left table- EVVERYTHING FROM 2ND TABLE IN QUERY/RIGHT and COMMON IN 1ST FULL JOIN: combines the results of both left and right outer joins- EVERYTHING The joined table will contain all records from both the tables and fill in NULLs for missing matches on either side. SELF JOIN: joins a table to itself as if the table were two tables, temporarily renaming at least one table in the SQL statement. CARTESIAN JOIN: returns the Cartesian product of the sets of records from the two or more joined tables.
